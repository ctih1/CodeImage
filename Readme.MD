# CodeImage
Make an image of a repository.

## Usage
``java -jar CodeImage.jar <username> <repo> <branch> <.filetype> <(optional) type>``

## Arguementes
- Username
  - Your GitHub username.
- Repo
  - Your repo **name**
- Filetype
  - The filetype (ex: .js)
- *Type*
  - maxHeight:
    - Prioritize the entire code.
  - minHeight:
    - Prioritize the 'look'
