## Usage
``java -jar CodeImage.jar <username> <repo> <branch> <.filetype> <token> <(optional) type>``

## Arguementes
- Username
  - Your GitHub username.
- Repo
  - Your repo **name**
- Filetype
  - The filetype (ex: .js)
- Token
  - GitHub personal access token. https://github.com/settings/tokens?type=beta
- *Type*
  - maxHeight:
    - Prioritize the entire code.

  - minHeight:
    - Prioritize the 'look'
